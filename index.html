☁️ Configuration stockage commandes (JSONBin)
Les commandes de tes clients doivent être stockées sur internet pour être visibles de partout. Ce service est 100% gratuit. Suis les étapes ci-dessous une seule fois.
Étape 1 — Créer ton compte JSONBin (gratuit)
1. Va sur jsonbin.io → Sign Up (gratuit)
2. Une fois connecté, clique sur "Bins" → "Create Bin"
3. Dans le champ JSON, colle exactement : {"orders":[]}
4. Clique sur Save → copie l'ID du Bin (ex: 64a1b2c3d4e5f60012345678)
5. Va dans API Keys → crée une clé → copie la Master Key
